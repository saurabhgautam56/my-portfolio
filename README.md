# My Portfolio Website

A modern and responsive personal portfolio website built using HTML, CSS, and JavaScript.
This portfolio showcases my projects, technical skills, certifications, coding profiles, and contact information in a clean and interactive UI.

---

## 🚀 Live Demo

🌐 Portfolio Website: https://saurabhgautam56.github.io/my-portfolio/

---

## 📌 Features

- Responsive modern UI
- Dark / Light mode toggle
- Animated skill progress circles
- Project filtering system
- Interactive certification cards
- GitHub live repository counter
- Scroll progress indicator
- Scroll reveal animations
- Mobile-friendly navigation
- Smooth scrolling experience
- Back to top button
- Coding profile dashboard
- Contact form UI
